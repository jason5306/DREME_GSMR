# DREME-GSMR

Source code for **Spatiotemporal Gaussian representation-based dynamic reconstruction and motion estimation framework for time-resolved volumetric MR imaging (DREME-GSMR)**.

## Overview

DREME-GSMR is a framework for dynamic reconstruction and motion estimation from MR data using a spatiotemporal Gaussian representation. It is designed for time-resolved volumetric MR imaging and supports both:

- dynamic reconstruction and training, and
- onboard real-time motion/image inference.
